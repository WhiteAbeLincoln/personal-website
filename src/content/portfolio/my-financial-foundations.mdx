# My Financial Foundations

A platform created for My Financial Foundations, LLC.

Consists of:

- Client-facing forms and general insurance information
- A system for insurance agents to track client progress, analyze data, and generate PDF reports.
- Editable content with NetlifyCMS, payment processing using Stripe, and infrastructure through AWS (Amplify, Cognito,
  Lambda, and DynamoDB).

Learn more from the [Financial Foundations website](https://myfinancialfoundations.com/).
