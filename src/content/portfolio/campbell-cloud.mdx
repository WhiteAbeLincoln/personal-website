# Campbell Cloud

A cloud platform for environmental data monitoring and management.

Features include:

- Remote & centralized monitoring of sensor networks
- Customizable dashboards for real-time and historical data display
- Automatically manipulate incoming data with user-created pipelines
- High-availability using reliable AWS infrastructure

Learn more from the [Campbell Cloud website](https://campbell-cloud.com/).
